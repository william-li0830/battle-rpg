# Milestone 1 — Core Game Skeleton

**Goal: Game runs + basic structure works**

 Create main game loop
 Create menu system (login / create / exit)
 Create Player class (username, password, health, points, score)
 Allow player to start a session
 
# Milestone 2 — Profile System (JSON)

**Goal: Players persist between runs**

 Create JSON storage for player profiles
 Save new player profiles
 Load existing profiles on login
 Validate login (username + password check)
 Update and re-save player data after changes
 
# Milestone 3 — Combat System

**Goal: Basic RPG fighting works**

 Create Enemy class (random health, attack, rewards)
 Create turn-based battle loop
 Implement player attack + enemy attack
 Add win/lose conditions
 Award score and points after battle
 
# Milestone 4 — Weapons System

**Goal: Weapons affect combat**

 Create Weapon class (name, damage, durability, price)
 Load weapons from JSON
 Equip weapon to player
 Apply weapon damage in combat
 Reduce durability per attack
 Handle broken weapons
 
# Milestone 5 — Question System

**Goal: Earn rewards outside combat**

 Create question bank
 Ask questions between levels
 Check answers
 Award points for correct answers
 Use points for upgrades
 
# Milestone 6 — Shop System

**Goal: Progression and upgrades**

 Create shop menu
 Buy health using points
 Buy weapons
 Prevent purchases without enough points
 Update player inventory/stats
 
# Milestone 7 — Levels & Progression

**Goal: Full game structure**

 Create 3 levels
 Increase enemy difficulty per level
 Trigger shop + questions between levels
 Track progression

**Goal: Game completion**

 Show final score screen

 Save final results
